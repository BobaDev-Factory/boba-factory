# Agent Catalog (Boba Factory)

## Leadership
- Owner (human): product/business authority and final validation.
- Main agent (orchestrator): coordination, gating, integration, final execution decisions.

## Core engineering agents
- Atlas: specification, scope, acceptance criteria.
- Forge-Backend: backend/API/DB implementation.
- Forge-Frontend: frontend/UI implementation.
- Sentinel: code review (bugs/security/performance/style).
- Weaver: browser E2E tests.
- Pulse: lint/tests/typecheck/smoke validation.
- Scribe: documentation/changelog/runbooks.

## Extended delivery agents
- Helm: release/ops, deployment readiness, rollback checks.
- Aegis: security hardening and risk analysis.
- Quarry: data/DB migration and rollback-safe changes.

## Product/business/creative agents
- Prism: product analysis.
- Mosaic: UX/UI design.
- Quill: content writing.
- Echo: social media.
- Vector: marketing.
- Radar: SEO.
- Beacon: analytics.
- Bridge: sales enablement.
